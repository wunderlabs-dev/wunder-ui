# Wunder UI

Design system built on [shadcn/ui](https://ui.shadcn.com) for Wunderlabs applications.

## Setup

```bash
pnpm install
```

## Development

```bash
pnpm dev
```

Opens Storybook at [http://localhost:6006](http://localhost:6006).

## Scripts

| Command      | Description                |
| ------------ | -------------------------- |
| `pnpm dev`   | Start Storybook dev server |
| `pnpm build` | Build static Storybook     |
| `pnpm lint`  | Run oxlint                 |
| `pnpm fmt`   | Run oxfmt                  |
